use command to test filelist_test.app
/am7x/bin/filelist_test.app arg1 arg2

arg1: it is the path where the file will be scaned, such as /mnt/udisk

arg2: if it is added, the file information will be shown, you can just fill it with 1

so for example:
/am7x/bin/filelist_test.app /mnt/udisk 1
will scan the udisk and show the information which is gotten
