Rotating and Resizing the Screen

The Alien application should inform the SecureWrx library when the screen's orientation changes to allow the SecureWrx library to present the document in a manner most suitable for the new aspect ratio.

Below is the usual default configuration of screen: the user is holding the device upright, and the entire screen is to be drawn by the SecureWrx library. There follows three other possible configurations of the device and the screen available for the SecureWrx library, each followed by a description of how the Alien application should handle transitions to and from the default state. This is not an exhaustive list of all possible configurations. It is a guide to common configurations from which the means of handling other configurations can be inferred.

The alien application should call PicselScreen_physicalRotation(). On some devices the area of the screen available to the alien application will change when the screen is pivoted, perhaps to keep a status bar on the top of the screen. On such devices, call PicselScreen_resize() with the new dimensions.
