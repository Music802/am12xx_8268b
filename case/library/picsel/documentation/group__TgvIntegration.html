Detailed Description

TGV supports document viewing, user interface and other media applications on embedded devices.

What is ENTWRX TGV?

Picsel's Truly Generic Version (TGV) is a package of Picsel's world-famous software in the form of a Software Development Kit (SDK) which allows convenient integration on to a wide range of operating system (OS) and device platforms without the need to wait for customisation by Picsel. It is based on software in C, and is delivered to Picsel's customers as a library which can be linked against an application.

The customer's application implements the initialisation and user interface, and makes calls to the SecureWrx library to handle the complex work of the product (see Developing an Application). It is called an "Alien application" for historical reasons, and is normally customised by software engineers of the device manufacturer (OEM) or system integrator, for statically linking into the device ROM during manufacturing. It is based on sample code that ENTWRX provides in this SDK.

A range of products and SecureWrxutions are available from ENTWRX using the TGV architecture, all of which share the benefits of the platform (see Benefits of TGV). The products are protected by copyright, patents, and other intellectual property rights.
