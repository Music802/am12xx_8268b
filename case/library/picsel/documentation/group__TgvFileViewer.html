ENTWRX File Viewer Development Kit is a world-famous document viewing product.

Delivering a range of documents and pictures to the broadest array of mobile devices does not always mean giving away control of your device's look and feel, being constrained in system design, or settling for second best. In a market of compressed timescales and branded differentiation, the ENTWRX File Viewer Development Kit (FVDK) allows manufacturers to design their own application around a feature-packed engine for content delivery. It is delivered as a linkable library with the core features for loading documents, displaying them on screen, and handling navigation through any user interface. It has already been shipped on hundreds of millions of devices.

FVDK empowers application designers to put state-of-the-art features in front of users, while retaining complete control of the appearance and capabilities. Features for document manipulation provided by the SDK include:

  • Load document from file, in memory, or from network.
  • Zoom in and out, fit page to screen, rotate.
  • Save screen-shot, Search for words or phrases.
  • Reformat large pages to fit on small screen.
  • Load large documents incrementally, to save memory.
  • Compatible with most European, Asian and other language regions.
