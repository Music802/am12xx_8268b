This page expands on Developing an Application and describes the settings and functions specific to File Viewer.

The sample applications can be referred to and used in porting the File Viewer to your platform.

Application Initialisation in File Viewer

In addition to The Main Flow of Control, you must implement the following:

  • If a document file has not already been selected, the Alien application should display a list of available files and wait for the user to choose one.
  • Call PicselApp_start() with Picsel_EntryPoint_FileViewer as the initFn parameter, and the file to be viewed described in the fileCommand parameter.
  • Set File Viewer specific settings in AlienConfig_ready(), see TgvConfigFileViewer.
  • The SecureWrx library will call AlienConfig_initialiseAgents(). This should activate the required Document Agents, for example by calling Picsel_Agent_Word() and Picsel_Agent_Html(). See TgvInitialiseAgents.
